ฉันต้องการเป็นทูตชุมชน

ส่งอีเมลถึงเราที่ hello@civi.me โดยใส่หัวเรื่อง "Ambassador Program" — หรือเยี่ยมชมหน้ามีส่วนร่วมเพื่อเรียนรู้เพิ่มเติมเกี่ยวกับบทบาทของทูตชุมชน

ฉันต้องการมีส่วนร่วมในการเขียนโค้ด

civi.me เป็นโอเพนซอร์ส เริ่มต้นด้วยการเรียกดูคลังข้อมูล GitHub ของเรา หรือส่งอีเมลถึงเราหากคุณต้องการหารือเกี่ยวกับแนวคิดในการมีส่วนร่วมก่อนเริ่มต้น

ฉันต้องการเป็นพันธมิตรกับ civi.me

เราทำงานร่วมกับห้องสมุด ศูนย์ชุมชน องค์กรด้านวัฒนธรรม มหาวิทยาลัย และองค์กรไม่แสวงหาผลกำไร ส่งอีเมลถึงเราที่ hello@civi.me โดยใส่หัวเรื่อง "Partnership"

ฉันพบข้อบกพร่องหรือมีคำขอเพิ่มฟีเจอร์

คุณสามารถเปิดประเด็นบน GitHub หรือส่งอีเมลถึงเราโดยตรง ทั้งสองวิธีใช้ได้

เกี่ยวกับ civi.me

civi.me เป็นโครงการของ Access100 ซึ่งเป็นองค์กรไม่แสวงหาผลกำไร 501(c)(3) ที่ตั้งอยู่ใน Honolulu, Hawaii ภารกิจของเราคือทำให้การมีส่วนร่วมของพลเมืองเข้าถึงได้สำหรับผู้อยู่อาศัยทุกคนใน Hawaii
