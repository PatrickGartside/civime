질문이 있으시거나, 참여하고 싶으시거나, 그냥 인사하고 싶으시든 언제든지 연락해 주십시오.

이메일 보내기

가장 빠른 연락 방법은 이메일입니다:

hello@civi.me

영업일 기준 5일 이내에 답변해 드리겠습니다.

자주 묻는 질문

앰배서더가 되고 싶습니다

hello@civi.me로 제목을 "Ambassador Program"으로 이메일을 보내시거나, 참여하기 페이지를 방문하여 앰배서더가 하는 일에 대해 자세히 알아보십시오.

코드에 기여하고 싶습니다

civi.me는 오픈 소스입니다. GitHub 리포지토리를 먼저 살펴보시거나, 기여 아이디어를 논의하고 싶으시면 이메일을 보내주십시오.

civi.me와 파트너십을 맺고 싶습니다

저희는 도서관, 커뮤니티 센터, 문화 단체, 대학교, 비영리 단체와 협력합니다. hello@civi.me로 제목을 "Partnership"으로 이메일을 보내주십시오.

버그를 발견했거나 기능 요청이 있습니다

GitHub에서 이슈를 등록하시거나 직접 이메일을 보내주십시오. 어떤 방법이든 괜찮습니다.

civi.me 소개

civi.me는 Hawaii Honolulu에 본부를 둔 501(c)(3) 비영리 단체인 Access100의 프로젝트입니다. 저희의 사명은 Hawaii의 모든 주민이 시민 참여에 접근할 수 있도록 하는 것입니다.
