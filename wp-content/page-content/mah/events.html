Event ko rekonaan itok

Ejjelok event ko rekonaan itok rebed er. Kim ej build-i am ambassador network kiiio — jokwe ilo ro mokta ro rej host ak attend-i juon civi.me event ilo am community.

Kwalok bwe kwoj interested

Ta kain event ko civi.me ej host-i?

Letter-writing party ko

Community gathering ko ia armij ro rej itok ibben doon nan jeje letter ko nan ri-kien ro aer. civi.me ej lewoj template ko, contact information, im submission guide ko. Ro rej itok rej boki ainikien ko aer im lale ko aer. Aolep issue ko, aolep lale ko rej welcome.

Letter-writing party ko remaroon jidik — jet jiran ilo juon coffee shop — ak event ko relaplap ilo library ko, community center ko, ak university campus ko. Kim ej lewoj grant ko jidik nan event ko nan jiban koba mona im supply ko nan event ko an ambassador ro.

Community information session ko

Workshop ko kin ewi wawein kwoj navigate-i public information ilo Hawaii: ewi wawein kwoj testify ilo juon hearing, ewi wawein kwoj lo meeting agenda ko, ewi wawein kwoj file-i juon public records request, im ewi wawein kwoj koba tool ko ilo civi.me. Mein ko rein rej practical, hands-on, im rej aim-i nan ro rej jino.

Ambassador meetup ko

Gathering ko nan civi.me ambassador ro nan connect ibben doon, share-i ta ko rejela, im plan-i event ko rekonaan itok. Open nan jabdewot armij eo ej interested in boki ambassador.

Kwoj konaan host-i juon event?

Kwoj jab aikuj boki official ambassador nan host-i juon civic engagement event. Elane kwoj konaan boki armij ibben doon kin juon community issue im kwoj konaan jiban im resource ko an civi.me, kim ekonaan rongorong jen kwe.
